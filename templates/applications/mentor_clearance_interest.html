You can volunteer with Girls of Steel without clearances as we have many tasks we need help with that don't involve the students. But if you would like to work directly with youth, we can help you start the process to get clearances. CMU will pay for the fees if you volunteer through us.

Pennsylvania law requires anyone working directly with children to have three background-check clearances on file. They are:

  • PA Child Abuse Clearance (PACA)
  • PA Criminal Record Clearance (PATCH)
  • FBI criminal fingerprint check

Are you interested in obtaining (or providing copies of) these clearances so you can work directly with our students?
